📝 Conceptos para el Reporte:
  • Integridad: La firma digital garantiza que solo aplicaciones autorizadas modifiquen datos
  • Autenticación de Servicio: El header X-API-KEY actúa como "sello" que identifica la app autorizada
  • Headers HTTP: Un lugar seguro para enviar credenciales sin exponerlas en la URL
  • Error 401: "No autorizado" - La firma es inválida o está ausente
  • Docker: Contenedor = Código + Dependencias = Reproducibilidad garantizada
